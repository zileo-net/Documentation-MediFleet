# Comment configurer un administrateur de bien sur un patient ?

> Pour configurer ce genre de gestion particulière, vous pourrez retrouver sur votre page de configuration, un sous menu à gauche de votre écran intitulé <a href=""> Gestions spécifique </a>.
>
>Pour un administrateur de bien spécifiquement, vous devrez créer une fiche client regroupant les informations pertinente de cet administrateur (adresse physique et/ou adresse mail pour l’envoi des factures), puis créer une nouvelle gestion spécifique en sélectionnant cette nouvelle fiche client dans le champs ‘Fiche d’institution cliente associée’ et en cochant la case ‘proposer l’envoi des factures vers cette institution cliente plutôt qu’au destinataire de la facture’.

>Après cela, vous pourrez depuis une fiche patient cliquer sur le bouton gestion particulière pour appliquer votre nouvelle gestion à un patient.
