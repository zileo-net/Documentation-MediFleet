# Comment retrouver le plus rapidement et le plus efficacement des données?

> L'accès le plus rapide à vos données est le petit cadre de recherche situé dans le bandeau supérieur bleu. Celui-ci permet de retrouver :
-Un patient, via son nom, NISS, ou adresse e-mail   
-Un utilisateur, via son nom ou son adresse e-mail      
-Une institution (client, sous-traitant...), via son nom  
-Un document de facturation, via son numéro  
-Une demande de transport, via un numéro de mission ou son identifiant Booqit



> Dans le cadre des recherches sur base des noms, vous ne devez pas respecter strictement la même utilisation des majuscules ou des accents, MediFleet s'accommodera de ces différences..


> Dans le cadre d'une recherche sur base d'un numéro, il est possible d'optimiser la recherche pour afficher moins de résultat. En utilisant juste une lettre indiquant ce que vous cherchez, suivi du numéro de document, vous obtiendrez des résultats plus précis.  
-"F" suivi d'un numéro pour chercher une facture  
-"M" suivi d'un numéro pour chercher une mission  
-"C" suivi d'un numéro pour chercher une note de crédit  
-"D" suivi d'un numéro pour chercher une facture    
> 
> Inutile de préciser l'année (indiquée dans votre numéro de document "version longue"), MediFleet proposera automatiquement les documents de l'année en cours et des 2 dernières années.
