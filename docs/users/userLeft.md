# Un collaborateur quitte l'entreprise, que faire?

> Si l'un de vos collaborateurs vous quitte l'entreprise, vous devez l'exclure de Medifleet. Il est important de ne pas laisser de compte utilisateur "ouvert" lorsque ce n'est plus nécessaire.
> 
> Pour cela, allez dans l'onglet "Relations" puis dans <a href=""> "Collaborateurs"</a> (ou toute autre catégorie de collaborateur). Cliquez sur la loupe bleue du correspondant au collaborateur en question, afin d'en afficher le profil. Vous trouverez sur ce profil un bouton rouge "Exclure" :
> 
> Utilisez ce bouton afin de désactiver le profil de l'utilisateur. Si le collaborateur a eu des interactions avec des missions ou factures, et donc des fiches patients, il n'est pas possible de supprimer totalement le compte utilisateur, celui-ci reste archivé sur votre MediFleet.
