# Documentation MediFleet

_Remarque : Ce repository est et doit rester public pour permettre l'affichage dans la documentation des captures d'écran._

## Lien public

La documentation est accessible ici : https://zileo-net.github.io/Documentation-MediFleet/

Après une ou plusieurs modifications des fichiers *.md dans ce respository, il y a lieu de "publier" ces modifications en allant dans l'onglet "Actions", choisir "Deploy Docsify" dans la colonne de gauche, puis exécuter l'action "Run Workflow".

## Modification de la documentation

Les fichiers représentant la documentation sont les `*.md` se trouvant dans les sous-dossiers de `docs`. Il suffit de les éditer directement dans GitHub. Il est possible de copier-coller des captures d'écran directement dans les pages (pour rappel, Maj+Command+4 sur Mac). Le reste de la mise en forme possible est accessible via le petit bouton "M" en bas à droite de chaque fenêtre d'édition.

<img width="1820" height="33" alt="image" src="https://github.com/user-attachments/assets/deb1a41e-6a31-4e47-a90f-abf802d5a489" />

Pour enregistre une modification, il faut utiliser le bouton vers "Commit changes..." en haut à droite de chaque fichier.

## Ajout d'une nouvelle page

Il suffit d'ajouter un document `.md` quelque part dans le dossier `docs` et d'indexer celui-ci dans le fichier `docs/_sidebar.md`.
